# NOVAKAYN AI

NOVAKAYN AI is a multilingual, AI-powered content analyzer that summarizes PDFs, videos, and academic materials into concise summaries, concept maps, and quiz questions.

🚀 **Features:**
- PDF and video input support
- Multilingual UI (9 languages)
- Auto-generated summaries, quizzes, and mind maps
- Responsive modern design

🔧 **Status:** Frontend MVP available – Backend and AI core under development.
